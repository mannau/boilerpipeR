Xerces 2.8.0 Plan

Features:

  • Start schema assessment from a specific type definition
  • Features for disabling ID/IDREF, unparsed entity and identity constraint checking
  • Enhancements to XML Schema annotation support
  • Support the warn-on-undeclared-elemdef feature which has been missing for many releases
  • Other misc. features, improvements and bug fixes

Target DCUT (code complete) date: February 24th, 2006

Target release date: February 27th, 2006
